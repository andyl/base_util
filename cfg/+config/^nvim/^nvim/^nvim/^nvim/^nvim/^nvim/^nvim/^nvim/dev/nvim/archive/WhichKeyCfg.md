# Which Key Config 

| Key     | File             |
|---------|------------------|
| NA      | map-base         |
| ,       | map-leader       |
| z       | map-z            |
| g       | map-g            |
| \       | map-slash        |
| [       | map-bracketopen  |
| ]       | map-bracketclose |
| <C-x>   | map-mod-c-x      |
| <A-x>   | map-mod-a-x      |
| <A-X>   | map-mod-a-X      |
| <C-A-X> | map-mod-a-x      |

Note: C- and C-A- only take lower case letters
Note: A- takes upper or lower case letters

| Group            | Prefix | Examples        |

| Terminals        | \t     | \tv \ts \tt     |
| Tabs             | ,t     | ,tn ,tx         |
| Xfile Copy/Paste | \x     | \xy \xd \xp     |
| Layout           | ,l     | ,le ,lr ,ls ,lv |
| Alt file nav     | \a     | \av \at \as     |
